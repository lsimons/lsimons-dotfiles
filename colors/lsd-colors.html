LSD Warm — source palette

OKLCH is the source of truth. Edit the OKLCH values below (format: L% C H — e.g. 64.39% 0.1373 66.7). Swatch, hex, and RGB update live; the hex value is what you mirror into zed/lsd-warm.json.symlink, ghostty/LSD-Warm-{Light,Dark}.symlink, and the clr[] table in oh-my-zsh/p10k.zsh.symlink. See README.md for the workflow.
